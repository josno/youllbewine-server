## Backend Server For A Wine App

### Technology Used

- Server Side Programming:

  - Javascript
  - Node
  - Express
  - Mongoose

- Database
  - MongoDB

### How to Use

    git clone https://github.com/josno/youllbewine-server.git

    cd youllbewine-server

    npm i

    npm start

### Endpoints

##### Wine Information || Available Methods:

###### GET, POST

`/api/v1/wine`
